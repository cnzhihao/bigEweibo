# 科学与迷信

> 发布时间：2020.10.10

[上一篇：年轻人的愤怒——学阀](/social/article2)

[下一篇：科学不需要信仰](/social/article4)



关于科学与迷信的关系我早就表达过，广义上一切迷信都是被信仰了的科学。所以，你们能从迷信中找出科学的成分一点不奇怪。

不过，科学这东西是什么就是什么。

 比科学少走一步是伪科学，那些迷什么易经五行中医之类的就属于这一类，死抱着陈旧已经被淘汰的知识当宝。

比科学多走一步也是伪科学，松鼠果壳之类的科普党属于这一类，成天牛逼乱吹把科学还在探索中的东西当做科学结论胡说八道。

你幼儿园的时候会写阿拉伯数字那叫数学挺好，你要是现在还只会写阿拉伯数字，就是白痴。两千年前你懂易经很牛逼，这年头还在研究易经的你不是脑残是啥?你要是真想研究万物之理，掌握自然规律，你特么不好好学物理，瞎几把折腾个啥?
