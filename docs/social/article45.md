# 关于法律大家必须得端正认识

>发布时间：yyyy.mm.dd

法律从不保护谁，它只是事后惩罚一下违法者；法律惩罚违法者也不是为了给你找回公平正义，而是为了维护它自己的尊严，维持社会秩序，保证自己的统治延续，跟你并没有多大关系。

比如法律会惩罚杀人犯，但是并不能保护你不被杀，当你已经是一个死人的时候对方受不受惩罚又与你何干？

学习法律不是要把一切交给法律，把自己的生命财产托付给警察、法官、律师，他们在绝大多数时候并不怎么靠谱...
