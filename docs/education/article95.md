# 沮丧是因为自己太矫情？

[上一篇：教育界的“拨乱反正”](/education/article94)

问：

大鹅有没有特别沮丧的时候？我一般是鸡血一点时间， 然后突然停下来就会有点感觉沮丧， 难受， 什么都提不起精神。 是不是我太矫情了？



答：

沮丧这种高端货一般是好学生的专利呀！ 像我这种从小学习很差，天天在班上垫底的滚刀肉，哪有资格沮丧？早就被练皮实了。
