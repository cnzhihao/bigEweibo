# 一边工作一抱怨的人，认清你自己

> 发布时间：yyyy.mm.dd

我周围也老有那么一些人,干着明明不爱干的工作,做一天和尚撞一天钟,出工不出力,美其名日“工作生活平衡”。

听着他们一边工作一边抱怨,我就纳闷:你这是耽误什么功夫呢?拜托认清一下你自己。

你之所以干这份工作是因为你蠢你没本事别的都干不了,什么生活所迫,这是生活给你机会好么!你的能力就残存下这么点了,你这是准备连这点都废掉彻底逼死自己吗?
