# 教育的目标—自主学习

[下一篇：教育界的“拨乱反正”](/education/article94)



教育，从小学到大学直至博士，终极目标只有一个：教会你自主学习，就像原子弹引信，只要把可持续的链式反应激发出来，就算大功告成。 

不过绝大多数人只是颗哑弹，当你毕业时庆幸终于不用再辛苦的学习的时候，你的废柴人生就被注定了。
